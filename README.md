# Simple ERC-20 Token

This repository contains a high-quality, production-ready implementation of an ERC-20 token on the Ethereum blockchain. It leverages the industry-standard OpenZeppelin libraries to ensure security and compatibility.

## Features
* **Standard ERC-20 Functions:** Transfer, Approve, and Allowance.
* **Minting:** Initial supply minted to the deployer's address.
* **Security:** Built using audited OpenZeppelin contracts.

## Quick Start
1. Install dependencies: `npm install`
2. Compile the contract: `npx hardhat compile`
3. Deploy: `npx hardhat run deploy.js`
